◆ Mission Briefing

ABOUT

Battleship is a free, open-source Android game implementing the classic naval combat experience. Deploy your fleet on a 10×10 grid, then hunt down the AI's ships before yours are destroyed. Fully offline — no account, no network, no ads.

◆ System Capabilities

FEATURES

◎

OFFLINE PLAY

No server, no network. Runs entirely on your device. Works anywhere.

⊞

10×10 TACTICAL GRID

5-ship fleet: Carrier, Battleship, Cruiser, Submarine, Destroyer.

◈

AI OPPONENT

Hunt/target algorithm with checkerboard parity — a real challenge.

▶

FLEET DEPLOYMENT

Place ships manually on the grid or use auto-deployment for fast starts.

◆ Installation

BUILD FROM SOURCE

# Clone git clone https://github.com/cocodedk/Battleship.git cd Battleship # Debug APK ./gradlew assembleDebug # Run tests ./gradlew test
